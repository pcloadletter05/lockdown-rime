```javascript
const EventBus = {
  _listeners: {},
  on(event, fn) {
    (this._listeners[event] ||= []).push(fn);
  },
  off(event, fn) {
    const arr = this._listeners[event];
    if (arr) this._listeners[event] = arr.filter(f => f !== fn);
  },
  emit(event, detail) {
    (this._listeners[event] || []).forEach(fn => fn(detail));
  }
};
```

Replace the placeholder content of `public/js/desktop.js` with a complete WindowManager implementation. This file defines the `WindowManager` global object.

**WindowManager singleton object with these properties:**
- `windows`: `new Map()` -- maps windowId to `{ element, title, icon, isMinimized, isMaximized, savedPosition }`
- `topZIndex`: starts at 100
- `cascadeOffset`: starts at 0, increments by 30 each window, resets when position exceeds half viewport width or half viewport height
- `activeWindowId`: null initially

**WindowManager methods:**

1. `createWindow({ id, title, icon, width, height, content })`:
   - Generate id if not provided: `'win-' + Date.now() + '-' + Math.random().toString(36).substr(2,5)`
   - Calculate position using cascade: `x = 50 + 30 * cascadeOffset`, `y = 50 + 30 * cascadeOffset`
   - Reset cascadeOffset when x > window.innerWidth / 2 or y > (window.innerHeight - 28) / 2
   - Increment cascadeOffset after positioning
   - Build DOM structure using `document.createElement()` (NOT innerHTML):
     ```
     div.nt4-window (position: absolute, left/top/width/height set via style)
       div.nt4-titlebar
         span.titlebar-icon (16x16 SVG if icon provided)
         span.titlebar-title (title text)
         div.nt4-titlebar-buttons
           button.nt4-titlebar-btn.minimize
           button.nt4-titlebar-btn.maximize
           button.nt4-titlebar-btn.close
       div.window-content (content HTML or empty)
       div.window-statusbar.well > span "Ready"
     ```
   - Attach drag handler to title bar via `initDrag(titleBar, windowEl)`
   - Attach click handlers to minimize/maximize/close buttons
   - Attach mousedown on entire window element to call `focusWindow(id)`
   - Append to `#desktop`
   - Store in windows Map
   - Call `focusWindow(id)`
   - Emit `EventBus.emit('window:created', { windowId: id, title, icon })`
   - Return the window id

2. `focusWindow(id)`:
   - If id is null or window not in map, return
   - Increment topZIndex, apply to window element
   - Set all windows' title bars to `.inactive` class
   - Remove `.inactive` from target window's title bar
   - Set `activeWindowId = id`
   - Emit `EventBus.emit('window:focused', { windowId: id })`

3. `minimizeWindow(id)`:
   - Set window element `display: none`
   - Mark `isMinimized = true` in the Map entry
   - If this was the active window, find the topmost visible window and focus it (by checking z-index of remaining visible windows). If no visible windows, set activeWindowId to null and emit `window:focused` with `{ windowId: null }`
   - Emit `EventBus.emit('window:minimized', { windowId: id })`

4. `restoreWindow(id)`:
   - Set window element `display: ''` (empty string to restore default)
   - Mark `isMinimized = false`
   - Call `focusWindow(id)`
   - Emit `EventBus.emit('window:restored', { windowId: id })`

5. `maximizeWindow(id)`:
   - If already maximized, call `unmaximizeWindow(id)` instead (toggle behavior)
   - Save current position/size in `savedPosition: { left, top, width, height }`
   - Set window to `left: 0, top: 0, width: 100vw, height: calc(100vh - 28px)` (28px = taskbar height)
   - Mark `isMaximized = true`
   - Add class `restore` to the maximize button (changes glyph to restore icon)
   - Call `focusWindow(id)`

6. `unmaximizeWindow(id)`:
   - Restore saved position/size from `savedPosition`
   - Mark `isMaximized = false`
   - Remove class `restore` from maximize button

7. `closeWindow(id)`:
   - Remove DOM element from `#desktop`
   - Remove from windows Map
   - If this was activeWindowId, find topmost visible window and focus it, or set activeWindowId to null
   - Emit `EventBus.emit('window:closed', { windowId: id })`

**Drag implementation** (private function `initDrag(titleBar, windowEl)`):
- mousedown on title bar (but NOT on `.nt4-titlebar-btn` -- check with `e.target.closest('.nt4-titlebar-btn')`)
- Calculate offset: `offsetX = e.clientX - windowEl.offsetLeft`, `offsetY = e.clientY - windowEl.offsetTop`
- `e.preventDefault()` to prevent text selection
- `document.body.classList.add('dragging')`
- `window.getSelection().removeAllRanges()`
- If window is maximized, unmaximize it first and recalculate offsets so drag feels natural
- mousemove on document: constrain title bar to remain on screen:
  - `newY = Math.max(0, Math.min(newY, window.innerHeight - 28 - 20))` (keep title bar above taskbar)
  - `newX = Math.max(-windowEl.offsetWidth + 100, Math.min(newX, window.innerWidth - 100))` (at least 100px of window visible)
- mouseup on document: `isDragging = false`, `document.body.classList.remove('dragging')`
- Also listen for `window blur` event to cancel drag if mouse leaves browser
